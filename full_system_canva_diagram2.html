Neural Architecture

Hidden Layer 1: 128 → 256 + ReLU
Hidden Layer 2: 256 → 256 + ReLU
Output Layer: 256 → 128 + L2 norm
Classifier: 256 → 6 relationship types
PyTorch neural network
Training & Loss Functions
Contrastive Loss: Pull related entities closer
Triplet Loss: Push unrelated entities apart
Relationship Loss: Classify relationship types
